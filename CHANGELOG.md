# 更新日誌（Changelog）

本專案的所有重要變更都會記錄於此檔案。

格式參考 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.0.0/)。

## [未發布] - 2026-05-31

### 同步播放 / 課堂模式
- 新增同步播放模式，自動判定主控端（master）與跟隨端（follower）角色。
- 主控者翻頁時所有 follower 畫面即時同步，並可顯示主控者的遠端游標位置。
- follower 加入需輸入 follower code，並追蹤線上連線人數。
- 主控者可鎖定 / 解鎖 follower 的音訊播放。
- master session 狀態持久化，重新整理後可接續。
- 課堂模式可設定每頁播完自動暫停，全螢幕顯示下一頁提示。
- 新增全螢幕 follower 提問對話框，可用 `s` 鍵切換顯示。

### 即時投票（Poll）
- 新增簡報進行中對當前頁發起即時投票。
- 支援以語音（WebAudio）建立投票，狀態跨 master / follower 即時同步。
- 主控者可控制投票開始 / 結束 / 重置，並切換是否對觀眾顯示結果。
- 即時顯示出席與作答人數。
- 新增全螢幕投票控制面板與鍵盤快捷鍵。

### AI 測驗（Quiz）
- 新增 AI 出題器（AI quiz builder）。
- 測驗模式跨端同步並追蹤每位學生作答、自動計分。
- 測驗播放時自動進入全螢幕。

### 分享與權限
- 新增分享 URL，區分唯讀 / 可編輯兩種權限。
- 分享對話框在無法自動複製時提供手動複製 fallback。
- 新增 PDF 擁有權與可見性存取控制。

### 認證
- 新增完整 Google OAuth 登入流程，可透過 `GOOGLE_AUTH_ENABLED` 旗標獨立開關。
- 驗證 `GOOGLE_CLIENT_ID` 是否為合法 OAuth Web Client ID，支援絕對與相對 redirect URI。
- HomePage 與 SettingsPage 顯示登入使用者資訊並提供登出。
- 新增 HTTPS 支援與憑證自動產生。

### AI 內容生成 / 編輯
- 新增以自然語言提示在現有簡報中新增多頁投影片（大綱 → 圖片 → 腳本 → 語音的完整流程），支援即時進度監看與中途取消。
- 投影片圖片編輯改用影像編輯 API，保留原版面僅調整指定部分。
- 新增基於 vision 的腳本重寫（OpenAI）。
- 新增渲染前的 AI 切分確認流程。
- 新增多輪 prompt outline 對話端點。
- 新增從 prompt 生成文字匯入。
- 新增每頁腳本長度限制設定。

### 匯入 / 匯出
- ZIP 匯出改為由後端直接串流，並支援首頁卡片一鍵匯出。
- ZIP 匯入會解析 `metadata.json` 還原頁數、狀態、進度等資訊。
- 匯入 / 上傳改用 `XMLHttpRequest` 顯示進度條並支援取消。

### 播放體驗
- 新增播放進度記憶與續播（以 localStorage 記錄頁碼與音訊位置，300ms debounce）。
- 改用真正的 HTML5 全螢幕 API。
- 手機播放時保持螢幕喚醒。
- 空白鍵改為切到下一頁；點擊投影片可切換播放 / 暫停。
- 顯示當前頁 QR code。
- 新增播放速度與字幕顯示設定。
- 音訊控制收合進狀態列。
- 漸進式載入腳本、預取縮圖並預載下一張投影片。

### 影片 / 音訊
- 影片生成改為顯示實際處理頁數進度（取代經過秒數），並改為直接下載生成的影片。
- 合成語音格式由 MP3 改為 M4A（AAC 128kbps，透過 ffmpeg 編碼）。

### 介面與首頁
- UI 與簡報內容語言可分別設定，大量介面完成國際化（i18n）。
- 首頁支援自訂分類新增 / 刪除與關鍵字標題搜尋。
- 視覺風格庫擴充（動漫、藝術渲染等選項）。

### 修正
- yt-dlp 與 ffmpeg 執行更穩健，並確保使用解析後的 Python 直譯器。
- 音訊來源切換時保留播放速率。
- PDF 處理中時防止對項目進行變更。
- 新增封面圖片的 cache busting 與靜態資源快取策略。
